Desenvolupa una aplicació en React per poder utilitzar a una empresa com a chat intern en temps real. 

Els usuaris han de poder registrar-se, iniciar sessió, enviar i rebre missatges i veure l'historial de converses emmagatzemades en una base de dades MongoDB.

Desenvolupat amb React + Vite o Create React App.

Ús obligatori de:

useState, useEffect, useContext, useRef i useReducer.

React Router per a navegació.

Context API per al maneig global de la sessió de l'usuari.

Components mínims:

Login, Register, Xat, MessageList, MessageInput i Users

Connexió amb backend mitjançant fetch.

Emmagatzematge del token de sessió en localStorage.

Ampliacions opcionals

Socket.io per a mensatgería en temps real.

Xat privat en un usuari concret.

Missatges vists i no vists
